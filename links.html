To define a link we use <a> tag.In this tag most important attribute is "href" it defines the destination of the link

The target attribute defines the where to open the linked files

there are four types of target values, _blank, _self, _parent, _top

target value is "_self" It is the deafult. Html Links

target value is "_blank" It opens the doc in new window. Html Links

target value is "_parent" It opens the doc in parent frame. Html Links

target value is "_top" It opens the doc in full body of the window. Html Links
